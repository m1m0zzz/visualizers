# WavGen Docs

[English Version](../wav-gen)

## 使い方

使用しているシンセに合わせて、サンプル数を選択してください。

Vital, Serumを使用している場合は、合わせて `lerp` を ON にして下さい。


## 仕様

### Preview

波形を表示します。

下のスライダーで、ウェーブテーブルポジションを変更できます。

### Config

- lerp

  線形補間を行います。

- normalize

  各フレームにおいて、ノーマライズを行います。

- frames

  `lerp` が ON の時のみ有効です。  
  線形補間後のフレーム数を指定します。


- filename

  ダウンロードするウェーブテーブルのファイル名です。

### Formula

内部的に expr-eval を使用しています。

expr-eval で使用できる修飾子、関数、定数のうち、変数への代入、`in` 修飾子以外のすべての機能を使用できます。

詳しくはこちらを参照してください。

https://github.com/silentmatt/expr-eval
